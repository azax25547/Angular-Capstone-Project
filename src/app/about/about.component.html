About This App

This is simply a Product Management App Which let the User to Manage Product Details

We provide various Sorting options so that Users can view the product from different necessity such as they can view the products by Quantity wise or views wise. Also we use global features like User form US can view the Product Prices in USD and we are working on bringing more filters.

Home

Our search implimentation helps user to find the specific product that he/she is looking for.

We provide the capabilities to the user to view the Products, Update them and Delete them.

We also follow secure access that no anynomous user can Modify the records and only registerd users can modify the records as their requirement.

We also brought a new feature Chart. This chart will help users to keep track in which Products are mostly viewed. We are working on this feature also for making it more user reliable and helpful.
